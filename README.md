# Simple todo list cli with Rust

Study purposed cli to manage todo list written in Rust.

### Features
1. add todo
2. list all todos, completed todos, open todos
3. update todo
4. mark todo done
5. undo marked todo
6. delete todo

### Demo

![todo-cli demo](demo.gif)

### Note
- To build & install CLI locally
```cargo install --path .```
- Or after code has changed
```cargo install --path . --force```

- To  run binaries
```cargo run --bin todo-cli```
```cargo run --bin todo-api```
